插入操作
a	：在光标当前位置后进行插入操作   
A	：在当前行的末尾进行插入操作
i	：在光标当前位置进行插入操作
I	：在当前行的开始进行插入操作
o	：在当前行的下面插入 空行
O	：在当前行的上面插入空行


光标移动
h	：左移一位
j	：下移一行
k	：上移一行
l	：右移一位
0	：移动到本行开头位置
Shift + 6（^）	：移动到本行第一个非空字符位置
Shift + 4（$）	：移动到本行末尾位置
w	：移动到下一个单词或标点符号的开头位置
Shift + w	：移动到下一个单词的开头位置，忽略标点符号
b	：移动到上一个单词或标点符号的开头位置
Shift + b	：移动到上一个单词的开头位置，忽略标点符号
Ctrl + f	：向下翻一页
Ctrl + b	：向上翻一页
n + G	：移动到文件的第n行
G   或者  Shift + g  	：移动到文件的最后一行


删除操作（剪切） 
x	：删除当前字符
n + x	：删除当前字符和之后n-1个字符
dd	：删除当前行
n + dd	：删除当前行和之后n-1行
dw	：删除当前单词，不包括标点符号
dW	：删除当前单词，包括标点符号
d + 0	：删除当前行起始位置到当前字符，保留当前字符
d + ^	：删除当前行起始非空位置到当前字符，保留当前
d + $	：删除当前字符至当前行末尾，删除当前字符
d + G	：删除当前行到文件末尾
d + n + G	：删除当前行到文件第n行


复制操作 
yy	：复制当前行
n  + yy	：复制当前行和之后n-1行
yw	：复制当前单词，不报扩标点符号
yW	：复制当前单词，包括标点符号
y + 0 	：复制本行起始位置到当前字符，不包括当前字符
y + ^	：复制本行起始非空位置到当前字符，不包括当前
y + $	：复制当前字符至当前行末尾，包括当前字符
y + G	：复制当前行至文件末尾
y + n + G	：复制当前行到文件第n行


粘贴操作 
p	：将缓存内的内容粘贴到光标之后
P	：将缓存内的内容粘贴到光标之前


查找操作
f + c	：在当前行中查找字符c，运行后输入；可重复执行
/ + chars	：在当前文件中查找字符串chars，输入n重复执行


替换操作
:%s/old/new/gc	：
	:			：用于启动一条ex命令
	%			：确定操作范围（本文件）可以用m,n表示
	s			：指定本次操作为 替换 操作
	/old/new	：搜索和替换的文本
	g			：全局，对每一行每一个实例进行操作，
	c			：请求用户确认
